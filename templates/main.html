Dog Breed Classifier App

This project performs 2 functions. It uses a SVM trained to 97% test accuracy to detect human faces vs dog faces. If a dog face is detected it then uses a keras model trained to 83% test accuracy to classify the breed of the dog out of 133 possible breeds, if a human face is detected it tries to classify the breed of dog the human resembles. Project source can be found here. Please test it out by uploading an image below and let me know what you think here at my Twitter. Or open an issue in the repo if you think there's something I can improve :)
